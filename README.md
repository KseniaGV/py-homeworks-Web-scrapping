# Парсер статей с Хабра

## Задание
Нужно парсить [страницу](https://habr.com/ru/articles/) со свежими статьями и выбирать те статьи, в которых встречается хотя бы одно из ключевых слов. Эти слова определяем в начале скрипта. Поиск вести по всей доступной preview-информации, т. е. по информации, доступной с текущей страницы.

## Формат вывода
Вывод в консоль список подходящих статей в формате: <дата> – <заголовок> – <ссылка>.
